JAK UŻYWAĆ (TL;DR):
================================================================================

- Po treningu: zaloguj trening do `data/training_log.csv`
  - Najłatwiej: opisz trening w czacie → asystent dopisze do CSV.
  - Szczegóły: `docs/instructions/10_logowanie_treningow.txt`

- Przed nowym planem (w nowym czacie): napisz „Daj mi plan” / „Wygeneruj plan” / „Plan na następny tydzień”
  - Plan jest ZAWSZE na 7 dni „rolling” = dzisiaj + 6 dni (nawet przy „następny tydzień”).
  - Szczegóły: `docs/instructions/20_generowanie_planu.txt`

- Sprawdź plan:
  - Podgląd: `index.html`
  - Źródło: `plans/current_plan.json`

- Workflow GIT (ważne):
  - Po KAŻDYM update:
    - planu (`plans/current_plan.*` lub `plans/archive/*`)
    - loga treningowego (`data/training_log.csv`)
    - wygenerowaniu nowego planu
    → asystent robi `git add -A`, `git commit`, `git push`.
  - Szczegóły: `docs/instructions/30_pliki_i_skrypty.txt`
